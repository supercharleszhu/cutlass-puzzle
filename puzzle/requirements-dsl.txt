# Dependencies for the CuTe DSL puzzles (days 10–14).
#
# Install with:
#   python -m venv .venv && source .venv/bin/activate
#   pip install --index-url https://download.pytorch.org/whl/cu124 torch  # see note
#   pip install -r puzzle/requirements-dsl.txt
#
# Notes:
#   - nvidia-cutlass-dsl is the official PyPI package for the CuTe Python DSL.
#     It is *separate* from `nvidia-cutlass` (the C++ template bindings).
#   - The DSL itself requires GPU compute capability sm_80 or newer
#     (A100 / H100 / B200 / RTX 30xx–50xx). It does NOT support sm_75 (T4) or
#     older.
#   - cuda-python: pick the line matching your NVIDIA driver below.
#       driver < 580 → cuda-python 12.x   (covers CUDA Toolkit 12.x)
#       driver ≥ 580 → cuda-python 13.x   (covers CUDA Toolkit 13.x)
#     Most production hosts on driver 550/570 want the 12.x pin.
#   - torch must be installed separately (it has its own CUDA bundle).
#     Pick the index URL matching your driver:
#       driver ≥ 550:  --index-url https://download.pytorch.org/whl/cu124
#       driver ≥ 570:  --index-url https://download.pytorch.org/whl/cu128
#       driver ≥ 580:  --index-url https://download.pytorch.org/whl/cu130
#     cu128+ is required if you want to run on B200 (sm_100).

nvidia-cutlass-dsl>=4.2
cuda-python>=12.0,<13      # change to >=13 only on driver ≥ 580
numpy>=1.24
# torch installed separately — see note above.
